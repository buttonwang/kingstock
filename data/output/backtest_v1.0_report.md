## 核心策略规则

- **买入**: RPS板块过滤 → MACD∩ZJTJ核心信号 → 增强规则过滤 → ML评分≥10
- **持有期**: ML评分13-15分→7天 / 11-12分→5天 / 10分→3天
- **卖出**: 仅按持有期到期自动平仓，不设价格止损/止盈/移动止盈
- **仓位**: Kelly仓位管理（13-15分100% / 11-12分75% / 10分50%）
- **市场状态**: 强势市场正常交易，震荡市场限制信号量
