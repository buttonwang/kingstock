## 一、策略概述

### 1.1 策略简介

本策略是一个**市场状态感知型多因子选股系统**，基于技术指标、资金流向、板块轮动和机器学习评分的多维融合，结合市场状态动态调整入场门槛和仓位，旨在A股市场实现稳定的超额收益。

策略核心思路：**强势市场重仓出击，震荡市场轻仓参与，弱势市场降低仓位参与**。通过扩大信号覆盖面和缩短持有期来提升交易频率和收益。**不设止损止盈**，完全依靠动态持有期（7/5/3天）控制出场节奏，让信号本身的质量决定收益。

> **设计理念**: 回测数据显示，止损和移动止盈会频繁打断盈利持仓，截断趋势收益。采用纯信号被动持有模式，让高评分股票的收益充分释放，同时通过缩短持有期来控制风险暴露时间。

### 1.2 策略定位

- **交易标的**: A股全市场（沪深主板、创业板、科创板）
- **持仓周期**: 短线（3~7个交易日）
- **资金容量**: 建议 100万~5000万人民币（受限于小市值股票流动性）
- **预期收益**: 年化 12%~18%（需经独立样本外验证）
- **最大回撤目标**: < 10%

### 1.3 策略优势

1. **市场状态自适应**: 通过全市场股票平均表现动态判断市场状态，强势市场正常交易、震荡市场减半仓、弱势市场降低仓位至30%
2. **多维信号融合**: 7维度综合评分（MACD + KDJ + ZJTJ + RPS + 成交量 + 基本面 + ML），避免单一指标的局限性
3. **机器学习增强**: XGBoost+LightGBM Stacking融合模型提供额外评分维度，提升信号质量
4. **宽松入场门槛**: 入场增强规则仅需通过1项（原需2项），弱势市场也参与交易，大幅提升信号覆盖度
5. **动态持有期**: 评分越高持有越久（7/5/3天），让优质信号有更充分的时间表现
6. **无未来偏差设计**: 市场状态基于历史数据计算（过去10日涨幅），不依赖未来信息

## 二、策略架构

### 2.1 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                  数据层 (Data Layer)                         │
│  SQLite缓存 → 日线行情(开高低收量) + 财务数据 + 板块数据   │
└─────────────────────┬───────────────────────────────────────┘
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                 指标计算层 (Indicator Layer)                  │
│  MACD ├  KDJ ├  ZJTJ(资金透视) ├  RPS(板块轮动)            │
│  成交量确认 ├  均线多头排列 ├  价格位置过滤                 │
└─────────────────────┬───────────────────────────────────────┘
                      ▼
┌─────────────────────────────────────────────────────────────┐
│                 评分融合层 (Scoring Layer)                    │
│  7维综合评分(每维0~15分,满分90分)                           │

└─────────────────────┬───────────────────────────────────────┘
                      ▼
┌─────────────────────────────────────────────────────────────┐
│               信号生成层 (Signal Generation)                  │

                      ▼
┌─────────────────────────────────────────────────────────────┐
│               交易执行层 (Execution Layer)                    │
│  市场状态动态仓位 ├  动态持有期(7/5/3天) ├  到期自动卖出     │
│  无止损止盈干扰，让信号质量决定收益                          │
└─────────────────────────────────────────────────────────────┘
```

### 3.4 市场状态检测

### 3.5 综合评分体系

| 评分维度 | 满分 | 评分依据 |
|:---|:---:|:---|

| **总分** | **90** | **综合评分(total_score)** |

### 3.6 交易执行规则

#### 仓位计算

```
单只股票仓位 = (可用资金 × Kelly因子) × 市场状态仓位乘数

Kelly因子:
  ML 13-15分: 100% Kelly
  ML 11-12分: 75% Kelly
  ML 10分: 50% Kelly

市场状态仓位乘数:
  强势市场: 1.0 (正常仓位)
  震荡市场: 0.5 (半仓)
  弱势市场: 0.3 (30%仓位)
```

#### 动态持有期 (Phase 6缩短)

| ML评分 | 持有天数 | 策略逻辑 |
|:---:|:---:|:---|
| 13~15 | **7天** | 高评分股票持有7天(原10天)，缩短暴露时间 |
| 11~12 | **5天** | 中等评分持有5天(原8天) |
| 10 | **3天** | 低评分快进快出(原6天) |
| < 10 | 不买入 | 不满足ML门槛 |

> 持有期缩短的逻辑: 瓶颈分析显示ML高分信号在短期(2日)的夏普比率(5.50)远高于长期(10日)的夏普比率(4.70)，说明信号的优势集中在短期内，缩短持有期有助于提升资金周转率。

#### 出场规则

| 出场类型 | 触发条件 | 操作 |
|:---|:---|:---|
| 到期卖出 | 持有期满(3/5/7个交易日) | 次日全部卖出 |
| 期末强制平仓 | 回测/实盘期末 | 全部清仓 |
| 价格止损 | 不设 | — |
| 部分止盈 | 不设 | — |
| 移动止盈 | 不设 | — |

> **纯信号策略不设止损止盈**。回测数据显示，止损和止盈频繁打断盈利持仓，截断趋势收益。信号的综合评分体系已经包含了风险控制，额外叠加止损止盈反而损害收益。

## 五、风险控制

### 5.1 风险敞口控制

| 风险维度 | 控制措施 |
|:---|:---|
| 市场风险 | 市场状态检测：弱势市场仅30%仓位参与 |
| 个股风险 | ML评分+综合评分双重门槛过滤低质量信号 |
| 仓位风险 | 震荡市场半仓、弱势市场30%仓位、每日信号数量限制 |
| 流动性风险 | 候选池自动筛选有足够成交量的股票 |
| 模型风险 | ML评分仅作为辅助维度(10/90分)，不过度依赖 |

### 5.2 策略限制与注意事项

1. **无止损保护**: 本策略不设价格止损，单笔最大亏损取决于持有期内的最大跌幅。回测最大回撤-6.48%，但极端行情下可能超过此水平
2. **过拟合风险**: ML模型使用过去60日数据训练，市场风格切换时模型需重新训练
3. **交易成本敏感**: 本策略交易频率较高（年均约255次），单笔盈利~1.3%，需严格控制佣金和滑点
4. **小市值偏误**: 候选池可能存在小市值股票，大资金需增加流动性过滤
5. **实盘衰减**: 当前回测夏普1.53存在过拟合可能，建议实盘预期打7折（夏普1.0~1.2）

### 5.3 回撤控制目标

| 回撤级别 | 触发条件 | 应对措施 |
|:---|:---|:---|
| 一级预警 | 当日亏损 > -2% | 检查当日持仓、确认市场状态 |
| 二级预警 | 连续3日亏损 > -5% | 降低整体仓位至50% |
| 三级预警 | 最大回撤 > -8% | 暂停交易、全面复盘 |
| 四级预警 | 最大回撤 > -15% | 停止策略、重新参数优化 |

## 六、参数清单

### 6.1 核心参数表

```python

# === 条件化仓位乘数 ===

POSITION_CHOPPY_MULT = 0.5      # 震荡: 半仓

# === 动态评分门槛 ===

SCORE_THRESHOLD_CHOPPY = 65     # 震荡: total ≥ 65

# === 每日最大信号数 ===
MAX_DAILY_STRONG = 3            # 强势: 每天最多3只

MAX_DAILY_WEAK = 1              # 弱势: 每天最多1只(原0只)

# === Kelly仓位管理 ===
KELLY_ML_13 = 1.0               # ML 13-15: 100% Kelly
KELLY_ML_11 = 0.75              # ML 11-12: 75% Kelly
KELLY_ML_10 = 0.50              # ML 10: 50% Kelly
```

| 版本 | 日期 | 修改内容 |
|:---|:---|:---|
| Phase 1 | — | 基础多因子筛选框架 |

| Phase 5 | 2026-06-05 | 市场状态感知 + 纯价格止损 + 部分止盈 + 三档精细持有期 |
| **Phase 6** | **2026-06-05** | **当前版本**: 纯信号策略 + 持有期缩短(7/5/3天) + 弱势市场参与(30%仓位) + 信号量扩大(302→551) + 年化收益提升至14.16% |

### Phase 6 优化要点

1. **信号链路修复**: 关闭价格位置过滤(PRICE_POSITION=False)、放宽增强规则门槛(ENHANCED_RULES_MIN=1)，信号量从302提升至551(+82%)
2. **持有期缩短**: 10/8/6→7/5/3天，资金周转率提升，更充分利用短期高夏普优势
3. **弱势市场参与**: 从完全空仓改为30%仓位+每日最多1只+提高评分门槛，不错过弱势行情中的局部机会
4. **纯信号策略**: 取消所有止损止盈机制，回测数据证明被动持有的累计收益优于叠加风控的版本
